DEMOQAWEB
1. Deskripsi Proyek
        Proyek otomasi pengujian berbasis Katalon Studio untuk form registrasi di situs DemoQA
        https://demoqa.com/automation-practice-form
        Mencakup:
   
        Pengujian data-driven menggunakan file Excel/CSV.
        Skenario positif 1 Test Cas (input valid). menggunakan data-driven
        Skenario negatif 17 Test Case (validasi mandatory field, format email, dll). menggunakan data-driven dan assert yang diperlukan.

3. Persyaratan
        Katalon Studio Enterprise v10.4.3 atau lebih baru
   
        Java JDK 8+
        Git untuk version control
        Akun GitHub (untuk sinkronisasi repository)
        File test data (Excel/CSV) sesuai struktur yang sudah disiapkan

5. Cara Membuka & Menjalankan Test Suite Collection
        Clone repository ini:
   
        bash
        [git clone https://github.com/username/DEMOQAWEB.git](https://github.com/sultanjoss/Katalon_QAtoolsWEB.git)
        Buka Katalon Studio → pilih Open Project → arahkan ke folder DEMOQAWEB.
        Pastikan semua Data Files sudah terhubung dengan Test Suite.
        Jalankan Test Suite Collection:
        Klik kanan pada Test Suites/TestSuiteCollection → pilih Run.
        Pilih browser target (Chrome/Firefox/Edge).
        Hasil eksekusi akan muncul di tab Reports.
